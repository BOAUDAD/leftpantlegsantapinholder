Data Table

Data tables are use to present raw data sets. Common uses are search results and reporting. In its simplest form it is able to show a top row of column names and rows for data.

Styles

Borderless Table

Borderless Tables are the default table in the design library. In its simplest form it is able to show a top row of column names and rows for data. They provide a good base to add features to after usability testing. Text is always left aligned in the back office. Numeric values are right aligned in most but only numeric values with a decimal are right aligned in Central.

Highlighted Table

On mouse over the row Tables are use to for sorting through large amounts of data. In Central the tables are borderless by default. To help improve the usability experience we have made the tables responsive to accomidate multiple screen sizes. The tables in central are always left aligned. In the Dev Ember Compontes the tables have been upgraded to allow for a selected state when cliecked while still allowing the hover over.
