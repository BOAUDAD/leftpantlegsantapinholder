Typography

Typeface

Open Sans is a clean, modern, legible font with many weights, which will help establish content hierarchy. It is also currently being used by other Q2 platforms. We will be using the below font weights in our application.

Open Sans Light (300)

Open Sans Regular (400)

Open Sans Semibold (600)

Open Sans Bold (700)


Font Stack

Using a CSS font stack is optimal in case Open Sans is not available. We recommend that the font stack specify Open Sans, Arial, Helvetica, then sans-serif.

Styles

Using rems for font sizing is beneficial because they behave predictably like pixels, but they are flexible enough to allow for responsive design. Sizing with rems will allow us to easily make widespread changes to the font sizing while maintaining the hierarchy of the content.
