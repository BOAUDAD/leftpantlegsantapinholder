Tabs

Dark Background Style

User tests were done to find if the visual style could be understood by an end user. The same component is being used in current prototype and still has not left a user confused or user on how to navigate through the page with the tab format.

The Central style uses a blue background with with white text. Hovering over the tab will cause the font to bold and maintain the bold style upon click. The indicator bar is set to 3 px, white, and lays under the selected tab. We have let the margin of the blue extend below the indicator so to not allow information below to bleed into the the tabs section.

A dark background on the tabs section became useful when working with tables. Many areas of the Central system have tables that were competing visually with the tabs. Setting the tabs to the same color of the header allow for the content of the table to be interpreted as a unique space for action.

Font weight of the unselected in 1/2 of the selected. In this example the font weight at selected in 600 and unelected in 300. A hover state will also appear at 600.

Light Background Style

Current designs for light background exist in prototypes only. They are considered to be a good practice for possible side bar actions. The current difference is design is that the colors invert. Text and the indicator go to blue while the background is set to white.
