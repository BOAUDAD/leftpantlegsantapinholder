General usage

Search is a nearly universal interface component that can be used to find one or multiple types of records.

Faceted search

Facets are different parts of a single type of record. When a user is given the opportunity to search through all records of a single type, consider using the faceted search modality to allow them to narrow their search.

For instance, a user record has a name, phone number, address, and type (retail or commercial) associated with it. Exposing these facets to the user allows them to easily refine their search.

Federated search

When a user can search across multiple types of related records (for instance, users, accounts, and customers) from one search box, use the federated search modality. Federated search offers the user the chance to either limit their search to one type of record out of many, or to simply let the search box decide what record types should be searched by default.
