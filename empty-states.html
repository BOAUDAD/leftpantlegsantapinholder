Empty states

Avoiding Empty States

Wherever possible, provide users with recommended content instead of defaulting to empty states. This helps the user quickly get started on a task.


On the Campaign List

Current is the default selection for the Campaign List. However, when there are no queued, active, drafted, or paused campaigns, or campaigns that need approval, the default option should become Recommended. This helps the user quickly take action on a campaign.


On the Dashboard

Similarly, while Campaign Performance is the default selection for the Dashboard, the default option should become Campaign Recommendations when there are no campaigns currrently running.


Displaying Empty States

If empty states can't be avoided, here are some tips to optimize them and create consistency:

  • Use icon colors that are in subtle contrast to the background so the icons do not appear actionable. (Our examples use #e0e0e0.)
  • Use large icons (120 x 120 pixels recommended) to maximize the use of space.
  • Instruct the user to take actions that will create content.
  • Use a positive, helpful tone.

On the campaign list

If the user selects Current when there are no campaigns, offer help on how to begin a campaign.


In the campaign builder

If there are no selected audiences, offer help on how to add them.


On the Dashboard

If the user selects Campaign Performance when there are no campaigns, offer help on how to begin a campaign.


Empty Search Results

If a search term does not return any results, provide recommendations for similar terms. Use suggested search terms that are actionable (clicking on them initiates the search), and look different than the actual returned results, so that users don't mistake them for actual results.




Acceptance Criteria

User testing:

  1. Users understand the instructions on the empty states.

Accessibility testing:

  1. Icon font characters are hidden from screen readers in all supported browsers.
  2. Text-to-background contrast ratio conforms to level AA.

Tech testing:

  1. Icons render properly in all supported browsers.
  2. When there is no active content available, recommended content is presented by default (where applicable).
  3. Colors and sizes match design guidelines provided here.
  4. Searches without results return actionable suggestions that initiate new searches.
